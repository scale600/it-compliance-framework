- [ ] **Add website link**:
  - Settings → General → Website
  - Enter: `https://project.techcloudup.com` (until GitHub Pages is live)

  - Branch: `main` → Folder: `/docs`

  - Update website link in repo settings to this URL

- [ ] **Pin to GitHub Profile**:
  - Go to your profile: `github.com/scale600`
  - Click "Customize your pins"
  - Select `it-compliance-framework` and check the box
  - Save

- [ ] **Add social preview image** (optional but high-impact):
